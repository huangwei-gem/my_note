# Jack叔叔说美国关税

---



## 这是怎样的一场崩盘?
起因:当时总统向全球加关税
大概的事件走向:
疯狂 →崩盘 →大萧条 →失业潮 →全球动荡
1929年10月，美国股市从历史高点 381点跌
1932年的 41点;相当于现在一万块变一千块。
跌幅高达 90%，财富几乎归零;
导致银行倒闭、企业停摆、失业暴增、全球经济崩溃。

## 到底发生了什么
1. 全民炒股、加杠杆
   只付 10%保证金就能炒股，10倍杠杆;
   理发师、送报童都在投机，市场泡沫极大。
2. 股市脱离基本面
   企业利润没增长，但股价暴涨;
   人人买入只是因为“还能更涨”
3. 一点风吹草动，就全面崩溃
   一开始只是少数股票波动，几天内演变为恐慌抛售。
---


四、后果有多严重?

| 指标    | 1929年  | 1932年    |
| ----- | ------ | -------- |
| 道指    | 381点   | 41点      |
| 失业率   | 3%     | 25%      |
| 银行倒闭  | 少量     | 9000家倒闭  |
| 工业产值  | 100%   | 减少一半     |
| 股票总市值 | $8750亿 | $1500亿左右 |


房子没了，工作没了，连银行存款也一夜清空
